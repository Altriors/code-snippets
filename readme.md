## 🚀 Quick Start

### 1️⃣ Clone the Repository
```bash
git clone https://github.com/Altriors/code-snippets.git
cd code-snippets
```

### 2️⃣ Navigate & Explore
```bash
# Browse available languages
ls snippets/

# Explore specific language snippets
cd snippets/python
ls -la
```
